## d == 3..d == 5 の動作
d == 3..d == 5 は 範囲演算子 (..) を使った条件式 で、これは フリップフロップ演算子 として機能します。

フリップフロップ演算子の動作:

- d == 3 が true になった瞬間に true を返し始める（開始条件）

- d == 5 が true になるまで true を返し続ける（終了条件）

- d == 5 が true になったら、その回の評価後に false に戻る
